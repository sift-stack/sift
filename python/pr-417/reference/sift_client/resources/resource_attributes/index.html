    create async ¶

    create(
    key_id: str,
    entities: str | dict | list[str] | list[dict],
    entity_type: int | None = None,
    resource_attribute_enum_value_id: str | None = None,
    boolean_value: bool | None = None,
    number_value: float | None = None,
) -> ResourceAttribute | list[ResourceAttribute]

    Create a resource attribute for one or more entities.

    PARAMETER DESCRIPTION
    key_id

    The resource attribute key ID.

    TYPE: str

    entities

    Single entity_id (str), single entity dict ({entity_id: str, entity_type: int}), list of entity_ids (list[str]), or list of entity dicts (list[dict]).

    TYPE: str | dict | list[str] | list[dict]

    entity_type

    Required if entities is str or list[str]. The ResourceAttributeEntityType enum value.

    TYPE: int | None DEFAULT: None

    resource_attribute_enum_value_id

    Enum value ID (if applicable).

    TYPE: str | None DEFAULT: None

    boolean_value

    Boolean value (if applicable).

    TYPE: bool | None DEFAULT: None

    number_value

    Number value (if applicable).

    TYPE: float | None DEFAULT: None

    RETURNS DESCRIPTION
    ResourceAttribute | list[ResourceAttribute]

    Single ResourceAttribute if entities is a single value, list of ResourceAttributes if it's a list.

    list async ¶

    list(
    *,
    entity_id: str | None = None,
    entity_type: int | None = None,
    key_id: str | None = None,
    include_archived: bool = False,
    filter_query: str | None = None,
    order_by: str | None = None,
    limit: int | None = None,
) -> list[ResourceAttribute]

    List resource attributes with optional filtering.

    PARAMETER DESCRIPTION
    entity_id

    Filter by entity ID.

    TYPE: str | None DEFAULT: None

    entity_type

    Filter by ResourceAttributeEntityType enum value.

    TYPE: int | None DEFAULT: None

    key_id

    Filter by resource attribute key ID.

    TYPE: str | None DEFAULT: None

    include_archived

    If True, include archived attributes in results.

    TYPE: bool DEFAULT: False

    filter_query

    Explicit CEL query to filter attributes.

    TYPE: str | None DEFAULT: None

    order_by

    How to order the retrieved attributes.

    TYPE: str | None DEFAULT: None

    limit

    How many attributes to retrieve. If None, retrieves all matches.

    TYPE: int | None DEFAULT: None

    RETURNS DESCRIPTION
    list[ResourceAttribute]

    A list of ResourceAttributes that match the filter.
