# Installation

`sift-cli` ships prebuilt binaries for macOS, Linux, and Windows. You can also
build it from source.

## Prebuilt installers (recommended)

Releases are published on GitHub with installer scripts generated by
[cargo-dist](https://github.com/axodotdev/cargo-dist). Download the installer
for your platform from the
[releases page](https://github.com/sift-stack/sift/releases) (release tags are
prefixed with `sift_cli-v`), or run the appropriate installer script.

**macOS and Linux:**

```sh
curl --proto '=https' --tlsv1.2 -LsSf \
  https://github.com/sift-stack/sift/releases/latest/download/sift_cli-installer.sh | sh
```

**Windows (PowerShell):**

```powershell
irm https://github.com/sift-stack/sift/releases/latest/download/sift_cli-installer.ps1 | iex
```

The installer places the `sift-cli` binary on your `PATH`.

## Build from source

You need a recent stable Rust toolchain. The build also generates this
documentation with [mdBook](https://rust-lang.github.io/mdBook/), so `mdbook`
must be on your `PATH`:

```sh
cargo install mdbook --locked
```

Then build and install the CLI:

```sh
git clone https://github.com/sift-stack/sift.git
cd sift/rust
cargo install --path crates/sift_cli
```

### Cargo features

`sift_cli` exposes the following Cargo features:

| Feature | Default | Description                                                                                                                                                                  |
| ------- | ------- | ---------------------------------------------------------------------------------------------------------------------------------------------------------------------------- |
| `mcp`   | off     | Adds the `mcp` subcommand (the Sift MCP server). The server is **in active development** — not built into default releases and not yet recommended for production use.       |

If you are exploring or contributing to the MCP server, you can opt in by
building from source with the feature enabled:

```sh
cargo install --path crates/sift_cli --features mcp
```

This is intended for development and evaluation only.

## Verify the install

```sh
sift-cli --version
```

If that prints a version, continue to [Configuration](./configuration.md) to
connect the CLI to your Sift environment.
